# ФерSat

### Запуск

Запуск делается командой `ghci main.hs`

### Пример использования

```
*Main> getQueensSet 4
"1 3; 2 1; 3 4; 4 2; "
```

**Если правильной расстановки ферзей не существует или передано отрицательное число, то выведется `" "`**
Пример:
```
*Main> getQueensSet 2
" "
*Main> getQueensSet (-1)
" "
```
